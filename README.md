# 口木科技

**如果你在你的服务器上修改了口木科技出现了错误请不要找我,作者不对修改版本负责**

口木科技是通过 RykenSlimefunCustomizer（简称 RSC）插件写成的，为粘液科技线带来了关于修仙等全新玩法。

安装口木科技需要使用 RykenSlimefunCustomizer（RSC），[点此下载 RykenSlimefunCustomizer](https://builds.guizhanss.com/SlimefunReloadingProject/RykenSlimeCustomizer/main)

需安装在在server(服务器根目录)\plugins（插件存放目录）\RykenSlimefunCustomizer（RSC插件配置）\addons（插件目录）

**在使用 `/rsc reload` 指令对口木科技进行热重载时请注意，如果出现机器报错请拆除后再次放置，如果还出现报错则需重启服务器。**

## 前置插件需求

前置：
- Slimefun
- GuizhanLibPlugin
- RykenSlimefunCustomizer

软前置：
- NetworkTechnology（网络附属，用于材质网桥相关物品）

## 提示

1. 若您之前使用其他方式加载过口木科技，请彻底清除旧配置后再使用本版本，以免冲突。

2. 口木科技包含有灵杖，卷轴，符箓等道具，建议服主根据服务器内容进行数值适配。

3. 如若您对口木科技中哪个道具感兴趣比如堆叠光暗等，这些为非公开脚本，请联系作者添加。

4. **注意**我希望口木科技不会参与到服务器内的买卖中（指大米而非金币）。

## 关于口木科技

口木科技由 口木A （Komu_A） 独立开发，旨在为粘液科技玩家提供修仙风格的扩展内容，包括灵石体系、法则悟道、身外身、灵杖与卷轴等丰富玩法。

未来将持续更新，目前需 Minecraft 版本（1.21+）。如果您有任何建议或反馈，欢迎提交至仓库 Issues，或练习作者：[2819696145]。
